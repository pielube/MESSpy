Here you have to set the input for your simulations.
You can modify file.json from notepad or from python.

For instructions on how to compile you can rad the _info.txt files that you can find in this folder.
Alternatively, we recommend reading comments in rec.py and location.py.
If you are interested in using a technology that is not present in input_test we suggest to see the comments in /techs/techname.py

If running MESS you have problem reading file.json probably you miss some comma or parenthesis.
In this case we suggest to use an online json validator to find the error.
